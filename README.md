# MagicSquare
![meguca](https://i.imgur.com/d1JdKWn.png)

Evolutionary algorithm solving different magic squares for an artificial life course.

## Building

Run `scons` anywhere

__or__

Run `simple_ga.sh` on Unix or `simple_ga.bat` on Windows
